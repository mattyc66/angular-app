Introduction to the Quote Generator

This project aimed to develop a quote generator for a department store to provide customers with personalized quotes for gift wrapping services. The system was designed with a graphical user interface to ensure a user-friendly experience, allowing customers to easily customize their gift wrapping preferences and receive an accurate price based on their choices. The application provides multiple options for the user to select from, including different present shapes, wrapping paper patterns and colours, and additional features like bows and gift card messages.

Building the Interface

The application was built in Python using the tkinter module, specifically CustomTkinter for a more modern visual style. The interface is organised using classes rather than a large collection of standalone functions, keeping related logic and widgets grouped together and easier to maintain.

Present dimensions are entered using spinboxes for width, height, and length, with radio buttons used to select the present's shape. The active spinboxes change depending on the selected shape: a cube only enables one spinbox, since all sides are equal, a cuboid enables all three, and a cylinder enables only width and height. A tkinter canvas is used to draw a live preview of the selected wrapping paper pattern and colour, updating immediately as the user changes their selection, alongside a dropdown for choosing between the six available colours.

Calculating the Quote

The required paper area is calculated based on the net of the selected shape, using the entered dimensions plus the 3cm clearance on each side to allow for overlap. This area is then multiplied by the price per cm² for whichever paper type is selected. A tick box enables an optional gift message, which also unlocks a text input for entering it, charged at a flat rate plus a per-character cost, and a separate tick box adds a bow at a fixed price. The total cost at the bottom of the interface updates live as any option is changed, giving the customer an immediate, accurate quote before finalising their order.
