Introduction to the Quote Generator

This project aimed to develop a quote generator for a department store to provide customers with personalized quotes for gift wrapping services. The system was designed with a graphical user interface to ensure a user-friendly experience, allowing customers to easily customize their gift wrapping preferences and receive an accurate price based on their choices. The application provides multiple options for the user to select from, including different present shapes, wrapping paper patterns and colours, and additional features like bows and gift card messages.

Requirements

Shape and Size input
Users must be able to selecr on of three diffrent shapes: Cube, Cuboid or Cylinder. The use must be able to input the width, length and height of the present, which will later be used to calculate the amount and price of the wrapping paper.



Wraping Paper Options

the system should provide the user with two diffrent paper types to choose from:

  • Cheap paper: costs 0.4p per cm²
  • Expensive paper: costs 0.75p per cm²

users can select between two diffrent pattern designs for the wrapping paper. user can also chose from six diffrent colours for the pattern design of the wrapping paper:

  • Purple
  • DarkSlateGray4
  • Deep Sky Blue
  • Light Sea Green
  • VioletRed2
  • Gold


Paper net calculation

The amount of wrapping paper required is calculated based on the net area of the selected present shape, with an additional 3cm clearance on each side to account for overlap. Using the dimensions entered by the user, the application determines the total net area of the present. This area is then multiplied by the cost of the chosen wrapping paper type to calculate the total price for the wrapping paper.



Additonal options

The application will provide the user with two additional options. a bow can be added for a fixed price of £1.50. a message can also be added with a flat rate of 50p along with an additional 2p per letter for the message.



Order summary

the application includes an option to generate a detailed sumary of the oder, which can then be saved to a text file. this file contains the dedtails of each item in the order along with the total cost of the quote.
