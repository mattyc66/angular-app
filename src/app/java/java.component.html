Introduction to the library loan system

The Library Loan System project was developed to simplify the management of library resources such as books and DVDs. Using Java and JavaFX, the system supports various loanable items, tracks availability, and manages loans and returns. Additional features include user authentication, overdue fine calculation, item searches, and detailed item reports. With an intuitive interface and object-oriented design principles, the application is tailored for internal library employees, ensuring ease of use and functionality.

System Design and Navigation

The application was built with a focus on seamless navigation and efficient functionality. A dynamic layout was implemented, allowing different FXML files to be displayed within the main view instead of navigating through multiple windows. This approach provided a smoother user experience while maintaining a consistent layout.

Tabs were used for organizing sections like books and DVDs, offering a clear and intuitive interface. Each page featured unique images to improve usability and visual appeal. To enhance functionality, the loans page included a date picker, allowing users to test overdue fine calculations by selecting custom return dates.

Refinements and Visual Design
